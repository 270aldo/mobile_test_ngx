## INSTRUCCIONES DE EJECUCIÓN

### Para cada imagen:

1. **Generar** usando el prompt exacto con Nano Banana Pro
2. **Revisar** que cumpla con la estética NGX (oscuro, premium, edad correcta)
3. **Ajustar** si es necesario (re-generar o editar)
4. **Exportar** en las dimensiones especificadas
5. **Nombrar** exactamente como se indica
6. **Guardar** en la carpeta correcta

### Orden de prioridad:
1. Hero images (BATCH 1) - Impacto visual inmediato en Home
2. Avatars (BATCH 2) - Necesarios para Chat y Coach
3. Empty states (BATCH 3) - UX de estados vacíos
4. Badges (BATCH 4) - Gamification

### Validación de calidad:

Cada imagen debe pasar estos checks:
- [ ] Fondo suficientemente oscuro para texto blanco encima
- [ ] Colores NGX presentes (violeta #6D00FF o mint #00F5AA)
- [ ] No se ve "stock photo genérica"
- [ ] Edad de personas es 30-50, no 20s
- [ ] Resolución correcta
- [ ] Formato correcto (JPG vs PNG)

## POST-PROCESAMIENTO

Después de generar todas las imágenes, ejecutar optimización:

```bash
# Optimizar JPGs para web/mobile
npx sharp-cli assets/images/hero/*.jpg --quality 85 --output assets/images/hero/

# Optimizar PNGs
npx pngquant --quality=65-80 assets/images/**/*.png --ext .png --force
```
